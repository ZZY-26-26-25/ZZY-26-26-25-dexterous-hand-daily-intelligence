# Dexterous Hand Daily Intelligence

具身智能与灵巧手研究情报库。

## 内容

- `reports/`：每日灵巧手情报报告
- `data/`：结构化索引（CSV/JSON）
- `docs/`：分类体系、数据规范、研究方法
- `scripts/`：自动更新与校验脚本

## 覆盖方向

- 灵巧手本体
- 遥操作与 UMI
- 数据采集手套与外骨骼
- 触觉与力反馈
- VLA / imitation learning / diffusion policy
- 数据集与开源工具链

## 更新规则

每次每日情报更新：

1. 新增 `reports/YYYY-MM-DD.md`
2. 更新结构化索引
3. 更新待跟踪列表
4. 更新采购与复现建议
