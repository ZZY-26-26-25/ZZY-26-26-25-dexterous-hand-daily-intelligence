# Awesome Dexterous Hand Catalog Design

## Goal

Build a searchable public knowledge base similar to Awesome-style research catalogs.

## Frontend

Recommended:

- React/Vite static website
- GitHub Pages deployment
- JSON driven filtering

## Filters

### Hardware

- DoF
- Actuation
- Direct drive / tendon / linkage
- Weight
- Cost

### Sensing

- Joint encoder
- Motor current
- Tactile
- Force torque
- Vision tactile
- Wrist tracking

### Data Collection

- UMI
- Wearable glove
- Exoskeleton
- Teleoperation
- Vision-only

### Software

- ROS2
- MuJoCo
- Isaac Lab
- LeRobot
- SDK availability
